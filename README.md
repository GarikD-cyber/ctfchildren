# Защитники Кибергорода

Образовательная квест-игра для детей 8–13 лет по информационной безопасности, цифровой гигиене и безопасному использованию искусственного интеллекта.

В проекте реализованы 24 интерактивные миссии, последовательное открытие по флагам, сохранение прогресса в `localStorage`, восстановление по флагу, шесть районов Кибергорода, финальный кодовый замок и персональный локальный бейдж.

## Локальный запуск

```bash
npm install
npm run dev
```

После запуска откройте адрес, показанный Vite в терминале.

## Сборка

```bash
npm run build
```

Проект не использует серверную базу данных, внешнюю аналитику или авторизацию. Имя, прогресс и достижения остаются в браузере пользователя.
